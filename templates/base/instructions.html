Quick use instructions:


  1. Ensure that in the current browser that you are logged out or logged into a facebook account with marketing data.
  2. On the home page select sign and and use facebook to sign into the web application.
  3. Once you are signed in and at the home page choose on of these three options to advance:
    • Select a date range and select submit.
    • Select 30 day report to generate a report for the last 30 days.
    • Select All Time Report to generate a report for all data.
  4. Select the Google Client you wish to report for and submit
  5. Select the Facebook Client you wish to report for and submit

    6. NOTE: At this point the Application gathers data from the APIs, this can take up to a few minutes.
  6. Data is shown for the selections made, the 'Download as Excel Sheet' button can be used to download an excel file with the data for the report summary.
